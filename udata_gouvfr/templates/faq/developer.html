Informations relatives à l’API

Le site Data.gouv.fr propose différents moyen d'accéder au catalogue des données :

  • une API complète, documentée avec Swagger
  • des dumps au format CSV, mis à jour en temps réel
    • catalogue des jeux de données: https://www.data.gouv.fr/datasets.csv
    • catalogue des ressources: https://www.data.gouv.fr/resources.csv
    • catalogue des organisations: https://www.data.gouv.fr/organizations.csv
    • catalogue des réutilisations: https://www.data.gouv.fr/reuses.csv
    • une liste des tags avec leur nombre d'occurences: https://www.data.gouv.fr/tags.csv
  • des statistiques de fréquentation anonymisées, mises à jour toutes les heures

L'intégralité des opérations réalisables depuis le site via votre navigateur peuvent être automatisées (publication d'un jeu de données, création d'organisations, etc). Veuillez consulter la documentation dédiée pour y accéder.

Moissonnage des dépôts de données ouvertes

Etalab est capable de moissonner des répertoires de données ouvertes selon plusieurs méthodes et formats de données :

  • les sites CKAN
  • les sites ayant ajouté les balises microdata Schema.org dans leurs pages HTML (plus précisément : les schémas DataCatalog et Dataset),
  • les sites publiant leurs données au format RDF Data Catalog Vocabulary (DCAT)
  • les catalogues spécialisés dans les données géographiques supportant le protocole Catalogue Service for the Web (CSW)


Si un dépôt ne supporte aucun de ces 4 formats, plusieurs solutions sont possibles, notamment :

  • Installer un CKAN devant le dépôt et y déployer un protocole de moissonnage supporté par le dépôt
  • Implémenter un sous ensemble de l'API CKAN, plus précisément les fonctions package_list, package_show et éventuellement organization_list et organization_show.

Note : Les moissonneurs réalisés pour le dépôt CKAN d'Etalab n'utilisent pas l'extension ckanext-harvest. Leur code source est libre et disponible sur GitHub.

Les partenaires techniques

data.gouv.fr est une plate-forme connectée à son écosystème. Des fournisseurs de plates-formes Open Data permettent aux acteurs économiques de gérer leur patrimoine de données publiques, qu'ils soient des collectivités, des instituts, des associations, des personnes morales privées ...

Nous listons ici tous les fournisseurs qui contribuent tous les jours à faire grandir le volume de données publiques en France :

  • OpenDataSoft
  • DataPublica
  • Easter Eggs
  • MGDIS (marque blanche)
