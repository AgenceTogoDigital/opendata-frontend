============
uData-gouvfr
============

uData customizations for Etalab / Data.gouv.fr.

Compatibility
=============

udata-gouvfr requires Python 2.7+ and uData.


Installation
============

You can install udata-gouvfr with pip:

.. code-block:: console

    $ pip install udata-gouvfr

or with easy_install:

.. code-block:: console

    $ easy_install udata-gouvfr
